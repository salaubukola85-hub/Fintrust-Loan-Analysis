# FINTRUST LOAN & REPAYMENT ANALYSIS
A complete end-to-end data analytics project using SQL and Power BI, focused on understanding loan distribution, repayment behavior, and default patterns across customer groups, employment types, and regions.

# Project Overview
This project analyzes:
Loan disbursement patterns,
Repayment performance,
Default rates,
Customer financial behavior,
High-risk segments and regions.
The dataset consists of three merged tables:
Customers,
Loans and
Payments.
SQL was used for data transformation and KPI calculations.
Power BI was used to build the dashboards.

# Tools & Technologies
SQL (joins, aggregations, KPIs)
Power BI (data modeling, dashboards)
